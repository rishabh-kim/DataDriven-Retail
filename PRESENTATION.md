# Sales Dashboard - Presentation

## Project Overview

This presentation covers the key highlights of the Sales Dashboard project, a comprehensive Tableau analytics solution analyzing 9,995+ sales transactions across 2014-2017.

---

## Slide 1: Executive Overview
**Sales Dashboard: Retail Performance Analytics**

- Analyzes 9,995+ sales transactions (2014-2017)
- Interactive dashboards for sales, profit, and year-over-year analysis
- Dynamic filtering by year enables trend identification
- Integrates sales, manager, and return data for complete visibility

**Business Value**: Transform raw transaction data into actionable insights for regional and category-level decision making.

---

## Slide 2: Sales Analysis Dashboard
**Key Highlights**

✓ **Regional Performance Breakdown**
- Sales by region with segment drill-down capability
- Identify high-performing and underperforming regions
- Track sales velocity by product category

✓ **Segment Analysis**
- Consumer, Corporate, and Home Office performance comparison
- Category preferences by customer segment
- Monthly and seasonal trends

✓ **Category Deep-Dive**
- Top and bottom performing product categories
- Sub-category performance within each category
- Sales contribution by category to total revenue

## Slide 5: Data Integration & Intelligence
**Comprehensive Data Model**

✓ **Three Data Streams Linked**
- **Orders Table**: 9,995 transactions with sales, profit, discount, quantity
- **People Table**: Manager-to-region mapping for accountability
- **Returns Table**: 297 returned orders linked to original transactions

✓ **Key Dimensions Available**
- Geography: Region, State, City
- Business: Segment, Category, Sub-Category
- Customer: Customer ID, Name, Segment
- Temporal: Order Date spanning 2014-2017

✓ **Derived Insights**
- Return rates by manager/region → Accountability
- Discount patterns by category → Margin protection
- Customer segment profitability → Strategic focus

---

## Slide 6: Interactive Capabilities
**How Users Engage**

✓ **Year-Based Filter**
- Parameter selector (2014, 2015, 2016, 2017)
- All dashboard views update dynamically
- Enables trend comparison and deep-dive analysis

✓ **Drill-Down Capabilities**
- Region → Category → Sub-Category
- Customer Segment → Product Detail
- Temporal trends with month-level granularity

✓ **Cross-Dashboard Navigation**
- Filter selections carry across dashboards
- Return patterns linked to manager performance
- Category performance visible across all analytical views

## Slide 8: Key Takeaways

**Problem Solved:**
Retail leadership now has real-time, consolidated visibility into sales performance across dimensions—replacing monthly static reports.

**Capability Delivered:**
- Dynamic year filtering for trend analysis
- Multi-dimensional drill-down (region, category, segment)
