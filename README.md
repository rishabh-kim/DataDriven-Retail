# Sales Dashboard

Interactive Tableau dashboards analyzing 9,995+ sales transactions across 2014-2017 to track performance by region, segment, and product category. The dashboard surfaces profitability trends, regional comparisons, and customer metricsâ€”enabling stakeholders to identify underperforming categories, top-performing managers, and return patterns through dynamic year-based filtering and drill-down capabilities.

## Key Features

- **Multi-dimensional Analysis**: Track sales and profit across regions, segments, and product categories
- **Dynamic Year Filtering**: Analyze trends year-over-year with interactive parameters (2014-2017)
- **Comprehensive Metrics**: Sales by category, profit analysis, and return pattern insights
- **Manager Performance Tracking**: Identify top performers and underperforming regions through People/Returns data linkage
- **Customer Insights**: Segment analysis across Consumer, Corporate, and Home Office

## Dashboard Views

1. **Sales Analysis Dashboard** - Regional sales breakdown with segment and category drill-downs
2. **Profit Analysis Dashboard** - Profitability trends and margin analysis by region and category
3. **Year Over Year Dashboard** - YOY comparison with trend analysis capabilities
4. **KPI Card Dashboard** - Key performance indicators at a glance

## Data Source

- **Dataset**: Sample - Superstore (9,995 orders)
- **Time Period**: 2014-2017
- **Key Tables**: Orders, People, Returns
